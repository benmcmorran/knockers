# knockers
![](iclauncher.png)  
Virtual doorbell system


## How to build

```
$ cd node  
$ npm i  
```

### Front-end

To compile static web pages.
```
$ cd node  
$ npm run webpack
```
```
$ cd node  
$ npm run watch
```

To serve static pages on localhost:8080.
```
$ cd node  
$ npm run serve
```

### Run Server

```
$ cd node  
$ npm run server
```
